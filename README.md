# expand-braille

- **what** it expands the dots from [unicode braille characters](https://en.wikipedia.org/wiki/Braille_Patterns#Block) to regular characters
- **why** cause you can't display braille characters on a tty
- **how** read on stdin or `-o <file>`. choose what character. '█' by default.

```
$ cargo r -- examples/amogus.txt
```
